# BBS-Proxy
日本向けに軽量化して作ったWEBProxyです。
## ~ 導入方法 ~
```curl -Ls https://raw.githubusercontent.com/uu5007mp4/BBS-Proxy/main/set.sh | bash```
### 終
注意としまして、192.168....のような内部IPでアクセスしてみたら私は失敗しましたので、注意してください。(私だけかもしれませんが)
